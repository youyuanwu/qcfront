# qcfront

**qcfront** is a Rust quantum computing framework built on
[roqoqo](https://github.com/HQSquantumsimulations/qoqo) for circuit
construction and [QuEST](https://github.com/QuEST-Kit/QuEST) (via
`roqoqo-quest`) for high-performance statevector simulation.

This site is the published documentation for the project. It is
generated with [mdBook](https://rust-lang.github.io/mdBook/) from the
sources in the `docs/` directory of the repository.

## Status

The book is just getting started — most of the project's design notes
currently live in the [`notes/`](https://github.com/youyuanwu/qcfront/tree/main/notes)
directory of the repository and will be migrated here as they are
polished.
